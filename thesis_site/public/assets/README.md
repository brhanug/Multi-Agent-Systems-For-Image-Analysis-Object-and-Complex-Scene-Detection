# Visual Historian: Multi-modal Historical Image Analysis Dataset

This dataset contains digitized historical photographs with multi-modal enhancements, generated as part of a Master's Thesis at the University of Hildesheim.

## Dataset Structure
- `images/original/`: Raw historical scans (grayscale).
- `images/restored/`: AI-restored images (CycleGAN + Real-ESRGAN).
- `labels/yolo_labels/`: Object detection labels in YOLO format (10 classes).
- `labels/vlm_captions/`: Detailed semantic captions generated by BLIP-2 and LLaVA-OneVision.
- `metadata/manifest.csv`: Master index of all images and their associated metadata.
- `metadata/kosmos_grounding.jsonl`: Rich Markdown descriptions (OCR + Layout) from Kosmos-2.5.
- `metadata/scene_graphs_v1.jsonl`: Knowledge graph triplets (Subject-Predicate-Object) from LLaVA.

## Taxonomy
### Object Detection (10 Classes)
person, child, horse, building, weapon, vehicle, tree, clothing, text, animal.

### Scene Classification (5 Categories)
teaching, family, playing, landscape, drawing.

## Usage
Refer to `metadata/manifest.csv` for mapping between image IDs and their semantic descriptions.
